# Introduction

Vapi is the developer platform for building voice AI agents. We handle the complex
infrastructure so you can focus on creating great voice experiences.

Build your first voice agent in 5 minutes using our dashboard.

## Ways to build

You have full control over each component, with dozens of providers and models to choose from.

### Dashboard

Create and configure an assistant in the Vapi dashboard, then test it in the browser.
No code required to get started.

### Web SDK

Add a voice agent to your web app with our client SDK. Embed a voice widget or build
your own interface.

### CLI

Install the Vapi CLI to manage assistants, phone numbers, and calls from your terminal.

```bash
curl -sSL https://vapi.ai/install.sh | bash
```

### Popular use cases

Explore what you can build:

- Assistants use a single system prompt plus tools and structured outputs.
- Squads let multiple assistants hand off to each other.
- Workflows orchestrate multi-step voice flows.

## Key capabilities

- Sub-600ms response times with natural turn-taking.
- Dozens of transcriber, model, and voice providers.
- Tools, structured outputs, and function calling.
- Inbound and outbound phone calls, plus web calls.
