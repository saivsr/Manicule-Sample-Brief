Quickstart

Create a voice agent and talk to it in your browser in about five minutes.

5 minUpdated July 2026
For agents

Building on Vapi with a coding agent like Claude Code or Cursor? Point it at AGENTS.md. It gives an agent the two calls and the rules it needs to get this working without guessing.

What you'll build

A voice agent that runs in your browser. You create the agent with one API call, connect to it with the Web SDK, and talk to it out loud. It takes about five minutes.

Before you start

You need a Vapi account and two keys from the dashboard:

  • your private key, used from your terminal to create the agent
  • your public key, used in the browser to connect to it
Note

The private key stays on your machine or server. Only the public key belongs in browser code.

1. Create an agent

Run this once. It returns an agent with an id that you use in the next step.

create-agent 
curl -X POST "https://api.vapi.ai/assistant" \
  -H "Authorization: Bearer $VAPI_PRIVATE_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support agent",
    "model": {
      "provider": "openai",
      "model": "gpt-4o",
      "messages": [{ "role": "system", "content": "You are Alex, a support agent for Acme." }]
    },
    "voice": { "provider": "11labs", "voiceId": "cgSgspJ2msm6clMCkdW9" },
    "firstMessage": "Hi, this is Alex from Acme. How can I help?"
  }'
const res = await fetch("https://api.vapi.ai/assistant", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.VAPI_PRIVATE_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Support agent",
    model: { provider: "openai", model: "gpt-4o",
      messages: [{ role: "system", content: "You are Alex, a support agent for Acme." }] },
    voice: { provider: "11labs", voiceId: "cgSgspJ2msm6clMCkdW9" },
    firstMessage: "Hi, this is Alex from Acme. How can I help?",
  }),
});
const assistant = await res.json();
console.log(assistant.id); // copy this id
Result

A JSON object with an id like "a1b2c3d4-...". Copy it for the next step.

2. Connect from your app

Install the Web SDK and create a client with your public key.

terminal
npm install @vapi-ai/web
app.js
import Vapi from "@vapi-ai/web";

const vapi = new Vapi("YOUR_PUBLIC_KEY");

3. Talk to it

Start the call from a click, so the browser can ask for microphone access. Pass the agent id from step 1.

index.html
<button onclick="vapi.start('YOUR_ASSISTANT_ID')">Talk to the agent</button>
Result

After you click and allow the microphone, the agent opens with its firstMessage, listens, and answers out loud.

What each part does

You'll usually touch three of the agent's fields, and you can change any one without touching the others:

  • model generates the replies. Swap gpt-4o for another provider or model.
  • voice sets how it sounds. Change the provider or voiceId.
  • the transcriber turns what the caller says into text for the model. Vapi uses a default until you set your own.
Shortcut

Just want it on a page with no build step? Paste the embed widget: a <script> tag plus a <vapi-widget public-key=… assistant-id=…> element, and the voice widget appears.
